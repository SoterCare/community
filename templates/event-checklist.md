# ✅ Event Checklist

The end-to-end checklist for any SoterCare event. Copy it into your event's planning issue and check items off as you go.

## 4+ weeks before

- [ ] Event proposal issue approved ([propose here](https://github.com/SoterCare/community/issues/new/choose))
- [ ] Date, venue/platform, and capacity confirmed
- [ ] Organizing team assigned (lead + 2-3 volunteers minimum)
- [ ] Budget / sponsorship needs identified
- [ ] Speakers confirmed — [Speaker Information](speaker-information.md) collected

## 2–3 weeks before

- [ ] Registration open (form or platform)
- [ ] Announcement posted (LinkedIn, Instagram, university channels)
- [ ] Venue logistics: projector, internet, power, seating checked
- [ ] Partner/sponsor logos and shout-outs confirmed

## 1 week before

- [ ] Reminder sent to registrants
- [ ] Speaker tech check done (slides, demos, connectivity)
- [ ] Volunteer roles assigned (registration desk, photos, timekeeping, tech support)
- [ ] Photo consent covered in registration or venue signage

## Event day

- [ ] Arrive early; set up and test everything
- [ ] Registration desk running; welcome attendees
- [ ] Photos taken (see [photo guidelines](../photos/README.md))
- [ ] Code of Conduct mentioned in opening remarks
- [ ] Feedback form shared before closing
- [ ] Thank speakers, volunteers, and partners publicly

## Within 1 week after

- [ ] Thank-you messages to speakers and partners
- [ ] [Event Report](event-report.md) filled and PR opened
- [ ] Photos and slides added to the repo
- [ ] Social media recap posted
- [ ] Feedback reviewed with the organizing team — lessons noted in the report
