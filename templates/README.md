# 📋 Templates

Ready-to-copy templates that keep our community's documentation consistent.

## Reports

| Template | Use after |
| --- | --- |
| [Event Report](event-report.md) | Any community event |
| [Workshop Report](workshop-report.md) | A workshop or bootcamp session |
| [Hackathon Summary](hackathon-summary.md) | A hackathon (hosted or attended) |
| [Meetup Notes](meetup-notes.md) | Informal meetups and discussions |
| [Speaker Information](speaker-information.md) | Collecting speaker details before an event |

## Checklists

| Checklist | For |
| --- | --- |
| [Event Checklist](event-checklist.md) | Any event, end to end |
| [Speaker Checklist](speaker-checklist.md) | Speakers preparing a session |
| [Organizer Checklist](organizer-checklist.md) | The person leading an event |

## How to use

Copy the template into the right folder (e.g. `events/2026-08-01-ai-workshop/report.md`), fill it in, and open a PR. Don't edit the templates themselves unless you're improving them for everyone.
