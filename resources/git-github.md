# 🔧 Git & GitHub

The foundation for everything we do. Learn these first — the rest of open source builds on them.

## Learn by doing

- [GitHub Skills](https://skills.github.com/) — interactive courses that run on GitHub itself; the fastest way to learn by doing
- [First Contributions](https://github.com/firstcontributions/first-contributions) — a practice repo built for your literal first pull request
- [Learn Git Branching](https://learngitbranching.js.org/) — visual, game-like way to finally understand branching and merging

## Reference

- [Pro Git book](https://git-scm.com/book/en/v2) — free and definitive; chapters 1–3 cover what most people need
- [GitHub Docs](https://docs.github.com/) — the official manual for everything GitHub
- [Oh Shit, Git!?!](https://ohshitgit.com/) — how to get out of common Git messes

## Cheat sheets

- [GitHub Git Cheat Sheet](https://education.github.com/git-cheat-sheet-education.pdf) — one-page PDF of the commands you'll actually use
