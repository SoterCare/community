# 🌱 Beginner Resources

A curated starting point for the tools and topics SoterCare works with. Everything here is free (or has a generous free tier). One rule for additions: say *why* it's worth someone's time.

## Git & GitHub (start here)

- [GitHub Skills](https://skills.github.com/) — interactive courses run entirely on GitHub; the fastest way to *do* Git rather than read about it
- [Pro Git book](https://git-scm.com/book/en/v2) — free, definitive; chapters 1–3 are all a beginner needs
- [First Contributions](https://github.com/firstcontributions/first-contributions) — a practice repo built for your literal first PR (then do your real one [here](../guides/first-contribution.md))

## Open Source

- [Open Source Guides](https://opensource.guide/) — GitHub's own guides; "How to Contribute" is essential reading
- [GirlScript Summer of Code](https://gssoc.girlscript.tech/) — the program we're part of; beginner-friendly on-ramp to real projects

## AI & Machine Learning

- [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) — practical, visual, free
- [fast.ai](https://www.fast.ai/) — top-down: build working models first, theory second
- [Kaggle Learn](https://www.kaggle.com/learn) — bite-sized hands-on courses with instant feedback

## Cloud Computing

- [AWS Educate](https://aws.amazon.com/education/awseducate/) — free student credits and labs (pairs with our AWS Cloud Club partnership)
- [AWS Cloud Practitioner Essentials](https://explore.skillbuilder.aws/learn/course/134/aws-cloud-practitioner-essentials) — the standard first cloud certification path

## IoT & Embedded

- [Random Nerd Tutorials](https://randomnerdtutorials.com/) — the ESP32 tutorials our IoT projects are built on
- [Edge Impulse docs](https://docs.edgeimpulse.com/) — TinyML the way we use it in SoterCare Studio

## Blockchain / Web3

- [Algorand Developer Portal](https://developer.algorand.org/) — official docs + tutorials from our workshop partner
- [Solana Developer Resources](https://solana.com/developers) — official starting point for Solana development

## Web & Mobile

- [The Odin Project](https://www.theodinproject.com/) — full-stack curriculum, project-driven, genuinely free
- [React Native docs](https://reactnative.dev/docs/getting-started) — our mobile stack's official tutorial
- [Next.js Learn](https://nextjs.org/learn) — our web stack, taught by its makers

## Healthcare Tech Context

- [WHO Digital Health](https://www.who.int/health-topics/digital-health) — the problem space we build for, from the source
