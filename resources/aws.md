# ☁️ AWS & Cloud

Cloud computing is a core part of the SoterCare stack. These resources take you from zero to certified.

## Start here

- [AWS Educate](https://aws.amazon.com/education/awseducate/) — free credits, labs, and courses for students, no credit card needed
- [AWS Cloud Practitioner Essentials](https://explore.skillbuilder.aws/learn/course/134/aws-cloud-practitioner-essentials) — the standard first course for cloud fundamentals

## Go deeper

- [AWS Skill Builder](https://skillbuilder.aws/) — AWS's official learning platform, hundreds of free courses
- [AWS Well-Architected Framework](https://aws.amazon.com/architecture/well-architected/) — how to design cloud systems the right way
- [AWS Free Tier](https://aws.amazon.com/free/) — build real projects within the always-free limits

## Certifications worth targeting

- **Cloud Practitioner** — foundational, great first cert
- **Solutions Architect – Associate** — the most recognized associate-level cert

## Community

Connect through the [AWS Cloud Club](../partners/aws.md) — a community SoterCare engages with for cloud learning.
