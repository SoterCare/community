# ❤️ Open Source

Open source is one of the best ways to learn software engineering — and a core part of the SoterCare community.

## Start here

- [Open Source Guides](https://opensource.guide/) — GitHub's own guides; "How to Contribute to Open Source" is essential reading
- [SoterCare First Contribution Guide](../guides/first-contribution.md) — our step-by-step onboarding
- [SoterCare Open Source Contribution Guide](../guides/open-source-contribution-guide.md) — the fuller picture: issues, PRs, reviews, etiquette

## Find things to work on

- [good first issue (across SoterCare)](https://github.com/search?q=org%3ASoterCare+label%3A%22good+first+issue%22+state%3Aopen&type=issues)
- [Good First Issue](https://goodfirstissue.dev/) — curated beginner-friendly issues across popular projects
- [Up For Grabs](https://up-for-grabs.net/) — projects with tasks set aside for newcomers

## Programs to join

- [GirlScript Summer of Code](https://gssoc.girlscript.tech/) — beginner-friendly; SoterCare is an [Ambassador & Contributor](../events/gssoc-2026.md)
- [Hacktoberfest](https://hacktoberfest.com/) — annual celebration of open source every October
- [Google Summer of Code](https://summerofcode.withgoogle.com/) — paid open-source contributions with mentoring

## Why it matters

Open source teaches you to read code, collaborate, take feedback, and ship — the exact skills employers look for. Every contribution is portfolio evidence.
