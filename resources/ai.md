# 🤖 AI & Machine Learning

AI powers much of what SoterCare builds in healthcare. These resources take you from fundamentals to shipping models.

## Start here

- [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) — practical, visual, and free
- [fast.ai — Practical Deep Learning](https://course.fast.ai/) — top-down: build working models first, theory second
- [Kaggle Learn](https://www.kaggle.com/learn) — bite-sized hands-on courses with instant feedback

## Go deeper

- [Andrew Ng's Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) — the classic foundational course (audit free)
- [Hugging Face Learn](https://huggingface.co/learn) — modern NLP, LLMs, and transformers, hands-on
- [3Blue1Brown — Neural Networks](https://www.3blue1brown.com/topics/neural-networks) — the best visual intuition for how neural nets work

## Edge & TinyML (our IoT work)

- [Edge Impulse Docs](https://docs.edgeimpulse.com/) — TinyML the way we use it in SoterCare's IoT projects

## Build with AI

- [Claude API / Anthropic docs](https://docs.anthropic.com/) — build AI-powered applications with Claude

## Why it matters

AI is reshaping healthcare — from diagnosis support to monitoring. Learning it well lets you build things that genuinely help people.
