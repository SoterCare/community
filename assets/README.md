# 🎨 Assets

Brand assets and graphics for the SoterCare community.

## What lives here

- `SoterCare-Primary-logo-brandblue.webp` — primary wordmark (light backgrounds)
- `SoterCare-Primary-logo-white.webp` — primary wordmark (dark backgrounds)
- Event banners, badges, and social graphics as they're created

## Usage

Use these assets to represent SoterCare at community events and in community materials. Don't alter the logo (colors, proportions, effects). For anything outside community use, ask the core team first via an issue.
