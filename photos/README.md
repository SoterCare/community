# 📸 Photos

Photos from SoterCare events, workshops, and hackathons.

## What lives here

- One folder per event: `YYYY-MM-DD-event-name/`
- Compressed images only (WebP or optimized JPEG, ≤500 KB each) — pick the best 10–20 shots per event, not the whole camera roll

## Consent matters

Only add photos taken at SoterCare events where attendees were informed photos would be shared publicly. If someone asks to have a photo removed, open an issue and we'll remove it promptly — no questions asked.
