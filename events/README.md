## Adding an event

1. Copy the [Event Report template](../templates/event-report.md) or model a page on an existing one (same section order)
2. Add photos to `photos/YYYY-MM-DD-event-slug/` and generate thumbnails
3. Link the new page from this index and the [timeline](../impact/timeline.md)

Read [How to Organize a Workshop](../guides/organize-a-workshop.md) or [Hackathon](../guides/organize-a-hackathon.md) before running your own.
