## Partnering with SoterCare

Interested in collaborating? Reach us via [LinkedIn](https://www.linkedin.com/company/sotercare) or open a [Community Suggestion](https://github.com/SoterCare/community/issues/new/choose). When a partnership is confirmed, we document it here using the [Partner Announcement template](../templates/partner-announcement.md).
